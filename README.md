# 🤖 Universal AI Booking & Scheduling Agent (n8n + Gemini)

مساعد حجوزات ذكي، ديناميكي، وعام مصمم لخدمة مختلف الأنشطة التجارية والخدمية (عيادات، مراكز تدريب، مكاتب استشارية، خدمات صيانة، وغيرها). يقوم النظام بأتمتة عملية الحجز بالكامل وبشكل فوري دون أي تدخل بشري طوال الـ 24 ساعة.

---

## 🚀 الميزات الرئيسية (Features)

*   **فهم طبيعي للغة (Natural NLP):** تحليل رسائل العملاء وفهم التواريخ والأوقات بديناميكية ذكية (✨) عبر نموذج **Google Gemini**.
*   **جدولة فورية:** ربط تلقائي بـ **Google Calendar** لإنشاء الأحداث وحجز المواعيد في ثوانٍ معدودة.
*   **إدارة البيانات:** فحص وتحديث بيانات العملاء وحالات الحجز مباشرة عبر **Google Sheets**.
*   **استجابة فائقة السرعة:** نظام تواصل فوري ومباشر يرسل ردوداً مخصصة للعملاء عبر **Telegram** في أقل من ثانيتين.
*   **نظام ذاكرة مدمج:** يمتلك الـ Agent ذاكرة قصيرة المدى لمتابعة سياق المحادثة مع العميل بذكاء.

---

## 🛠️ هيكلية سير العمل (Workflow Architecture)

يتكون سير العمل داخل بيئة **n8n** من العقد (Nodes) التالية المتصلة ببعضها بشكل متكامل:
1.  **Telegram Trigger:** لاستقبال رسائل العملاء اللحظية بشكل تلقائي.
2.  **Get row(s) in sheet:** لقراءة والتحقق من بيانات العميل أو جدول المواعيد الحالي.
3.  **AI Agent (Main Core):** العقدة المركزية التي تدير المحادثة وتتخذ القرارات وتستدعي الأدوات.
    *   *الموديل المستخدم:* **Google Gemini Chat Model** عبر الاتصال الرسمي بالـ API.
    *   *الذاكرة:* **Simple Memory** للحفاظ على سياق الحوار.
4.  **Create an event in Google Calendar:** أداة (Tool) يستدعيها الذكاء الاصطناعي لجدولة الموعد ديناميكياً فور موافقة العميل.
5.  **Send a text message (Telegram Node):** لإرسال الرد الذكي والنهائي المنبثق من الـ AI إلى هاتف العميل مباشرة.

---

## 📋 المتطلبات المسبقة (Prerequisites)

قبل تشغيل المشروع، ستحتاج إلى تجهيز الحسابات والبيانات التالية:
*   حساب على منصة **n8n** (Cloud أو سيرفر VPS خاص مدمج بـ Docker).
*   **Telegram Bot Token** و **Chat ID** (يمكن الحصول عليهما عبر BotFather).
*   حساب **Google Cloud Console** مفعّل ومربوط به صلاحيات الوصول لـ:
    *   Google Calendar API
    *   Google Sheets API
*   مفتاح سري (API Key) من **Google AI Studio** لتشغيل نموذج Gemini.

## ⚙️ طريقة التثبيت والتشغيل (Setup Instructions)

1.  قم بتحميل ملف الـ `workflow.json` المرفق في هذا المستودع.
2.  افتح واجهة التحكم الخاصة بك في **n8n**.
3.  من أعلى يمين الشاشة، اضغط على الخيارات واختر **Import from File** ثم قم برفع ملف الـ `.json`.
4.  قم بفتح العقد وتوصيل بيانات الاعتماد (Credentials) الخاصة بك لكل من (Google Auth, Telegram Bot API, Google AI Studio).
5.  في عقدة `Create an event in Google Calendar`، قم بتحويل الحقل الخاص بالتقويم إلى `From List` واختيار تقويمك الرئيسي، مع إبقاء حقول الوقت مفعلة عبر وضع الذكاء الاصطناعي (✨).
6.  اضغط على **Publish** لتفعيل النظام في الخلفية بشكل دائم ومستمر!
