# 🤖 Telegram AI Booking Agent

نظام أتمتة ذكي ومتكامل لإدارة وتأكيد الحجوزات والمواعيد بالكامل عبر **تليجرام** دون أي تدخل بشري، مصمم ليعمل كـ Production-ready agent للشركات والعيادات لتقليل المهام الروتينية وزيادة كفاءة المبيعات بالاعتماد على منصة n8n.

## 🚀 الميزات الرئيسية (Key Features)
* **محادثات ذكية (AI-Driven):** معالجة اللغات الطبيعية وفهم نية العميل (Intent Detection) باستخدام نماذج الـ LLMs للرد بدقة ومرونة.
* **مزامنة المواعيد لحظياً (Real-Time Sync):** فحص الأوقات المتاحة وتحديث قاعدة البيانات فوراً لمنع التضارب (Double Booking).
* **إرسال تذكيرات مؤتمتة (Auto-Reminders):** نظام مدمج لجدولة رسائل التذكير وتأكيد الحجز تلقائياً لتقليل نسبة عدم الحضور.
* **إلغاء وتعديل مرن:** إمكانية تغيير الموعد أو إلغائه من قِبل العميل مباشرة عبر البوت وتحديث النظام فوراً.

## 🛠️ البنية التقنية (Tech Stack)
* **محرك الأتمتة (Orchestration):** منصة **n8n** لبناء وإدارة مسارات العمل.
* **بوابة المراسلة (Bot Gateway):** واجهة **Telegram Bot API** الرسمية لاستقبال وتوجيه الـ Webhooks.
* **معالجة الذكاء الاصطناعي (AI Core):** تكامل مع نماذج الذكاء الاصطناعي (مثل Google AI Studio).
* **قاعدة البيانات والجدولة:** **Airtable** أو Google Calendar لتخزين المواعيد وبيانات العملاء.
